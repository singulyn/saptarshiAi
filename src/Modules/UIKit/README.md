# UI Kit Module

Internal reusable AdminLTE v4 component catalogue for SaptariX Admin MVC developers.

Routes are exposed under `/UIComponents` and protected by `Developer.UIComponents.View`.

The module contributes menu entries, permissions, controller actions, and Razor pages. Reusable partials live in the Admin MVC host under `Views/Shared/UIComponents`, and page-level helpers live in `wwwroot/css/ui-kit.css` and `wwwroot/js/modules/ui-kit.js`.
